## Random Guesser
A Yr.10 python project of mine where I created a guessing game. User has to try to guess a number between 1 and 100. Defensive Design included, to prevent any errors to the program.
# Features
    *Time delays used for a smoother reading for the user instead of a block of text just outputted straight away.
    *Defensive design used to prevent use of strings for a integar input.
    *Range filtering used to help user to pick an actual number between 1-100
    *Seperates genuine attempts to guess number from a failed attempt.
# How to Play
1. Run this script into Python
2. Try guessing the number using the clues for help
